# Freaks of Nature

A modern, responsive web application for an electronic music and arts festival brand. Built with React, this platform provides an immersive experience for festival-goers with features including event information, ticket purchasing, content streaming, and community engagement.

## Features

- **Home Page**: Interactive countdown timer, event highlights, and featured content
- **Event Information**: Comprehensive event details, ticket options, and FAQ section
- **Relive Platform**: Access to archived DJ sets and festival moments
- **Blog Section**: Latest news, updates, and festival-related content
- **About Page**: Festival history, community information, and partnership opportunities
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Interactive Navigation**: Dynamic dropdown menus with smooth animations
- **Custom Animations**: Lottie-powered visual effects and transitions

## Technologies

- **React 18.3** - UI framework
- **React Router 6** - Client-side routing
- **Bootstrap 5** - Responsive layout and components
- **Lottie Animations** - Interactive animations
- **React Icons** - Icon library
- **Custom Fonts** - Stolzl font family

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd freaks-of-nature
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

## Available Scripts

### `npm start`
Runs the app in development mode. The page will reload automatically when you make changes.

### `npm run build`
Builds the app for production to the `build` folder. The build is optimized and minified for best performance.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run eject`
**Note: This is a one-way operation. Once you `eject`, you can't go back!**

Ejects from Create React App, giving you full control over the build configuration.
